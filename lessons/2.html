La création

Avant d'encoder notre base de donnée, nous devons tout d'abord créer une structure viable et coordonnée.

Dans le but de concevoir notre site de e-commerce, nous devons créer plusieurs tables :

  • Une table pour les produits
  • Une table pour les utilisateurs
  • Une table pour le panier
  • Une table pour les commandes

Quel language est necessaire pour créer une base de donnée ?

SQL est la bonne réponse et maintenant que nous avons une structure pour notre base de donnée, commençons sa création.

Exécutez la ligne suivante
CREATE TABLE utilisateurs (id INT, nom VARCHAR( 100))
Exécuter

Bravo, vous venez d'exécuter votre première requête SQL. Cette dernière a crée une table.

La voici :

Champ Type Null Clé Défaut Extra

Cependant cette dernière ne semble pas adapté à nos besoin, que pourrait-il manquer ?

La clef primaire est la bonne réponse, en effet, chaque table doit avoir une clef primaire pour différencier chaque ligne.

Mais l'incrémentation auto est aussi une bonne réponse, elle permet d'incrémenter automatiquement la clef primaire.

Maintenant, à vous de jouer pour créer toute la table !

Elle doit contenir les élement suivant :

  • Un id (avec les attributs PRIMARY KEY et auto_increment)
  • Un nom
  • Une date d'anniversaire
  • Un email
  • Un mot de passe
Exécutez la ligne suivante
CREATE TABLE utilisateurs ( [...], creation DATETIME NOT NULL DEFAULT CURRENT_TIMESTAMP)
Exécuter

Bravo, vous venez de créer votre première table tout seul !

La voici :

Champ Type Null Clé Défaut Extra
